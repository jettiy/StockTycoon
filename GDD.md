# 📊 주식잡스 (StockTycoon) — GDD

> 증권가에 취직한 청년이 주식/코인으로 돈을 벌고, 집·차·결혼까지 이루는 방치형 게임
> 카이로소프트 감성 + JRPG 픽셀 아트 · 엔진: Godot 4.7 (GDScript)

---

## 🎮 핵심 루프

```
주가 확인 → 매수/매도 → 수익 → 라이프 업그레이드 → 더 큰 투자 → 세대교체
                ↑                                              ↓
                └──────── 자동매매 + 오프라인 보상 ────────────┘
```

**핵심 재미 요소:**
1. 주가가 실시간으로 움직이는 관찰의 재미
2. 타이밍 맞춰 사고팔기 (도박적 쾌감)
3. 자산이 불어나면 라이프가 업그레이드되는 성취감
4. 오프라인 동안에도 자동매매로 돈 버는 방치형 쾌감

## 📈 주식 시장

### 16종목 구성

| 카테고리 | 종목수 | 특징 |
|---------|--------|------|
| 🇰🇷 한국 | 5 | 블루칩 중심, 안정적 |
| 🇺🇸 미국 | 7 | 성장주 비중 높음, 중변동 |
| 🪙 코인 | 4 | 고변동, 고위험고수익 |

### 가격 변동 모델

```
가격변동 = 랜덤워크 + 트렌드 + 마켓사이클 + 뉴스이벤트
```

- **랜덤워크**: 종목별 변동성(volatility) 적용 정규분포 난수
- **트렌드**: 장기 상승/하락 성분 (성장주 > 블루칩)
- **마켓사이클**: 사인파 기반 bull/bear 사이클 (코인이 가장 민감)
- **뉴스이벤트**: 특정 종목에 가격 충격 (감쇠하며 적용)

## 🏠 라이프 시스템

### 주거 (7단계)
고시원 → 월셋 → 원룸 → 투룸 → 아파트 → 펜트하우스 → 섬

### 차량 (7단계)
자전거 → 중고차 → 준중형 → 세단 → 스포츠카 → 슈퍼카 → 헬기

### 직급 (8단계)
신입사원 → 대리 → 과장 → 차장 → 부장 → 임원 → CEO → 전설의 투자자
(순자산 달성 시 자동 승진, 7일마다 월급 지급)

## 🔑 핵심 시스템

### 정보력 (에너지)
- 매매는 자유롭게 가능
- **뉴스 조회에 정보력 소모** (회복: 시간 경과 / 가구 버프 / 결혼 버프)

### 자동매매 슬롯
- 조건 설정: "가격이 X 이하면 매수 N주" / "수익률 X% 달성 시 매도"
- 오프라인에도 실행됨
- 슬롯 확장: 직급/결혼/NPC 호감도로 해금

### 초반 난이도 완화
- 첫 3거래 무조건 수익 (튜토리얼)
- 첫 7일 수수료 50% 할인
- 파산 방지 지원금 (순자산 50만원 이하 시 200만원 지급)

### 코인 리스크
- 해킹: 보유 코인 일부 증발
- 거래소 폐업: 특정 코인 거래 중단
- 금리 인상: 코인 전반 하락
- 고래 덤핑: 특정 코인 급락

### 세대 교체 (New Game+)
- 자녀에게 자산 50% + 스탯 30% 상속
- 무한 세대 진행 가능
- 세대별 업적/랭크 추적

## 💻 기술 구조

```
StockTycoon/
├── project.godot              # 프로젝트 설정
├── data/
│   ├── stocks.json            # 16종목 데이터
│   ├── balance.json           # 밸런스 수치
│   ├── npcs.json              # NPC 데이터 (TODO)
│   └── events.json            # 이벤트 데이터 (TODO)
├── scripts/
│   ├── autoload/              # 싱글톤
│   │   ├── GameManager.gd     # 플레이어 상태
│   │   ├── MarketSim.gd       # 주가 시뮬레이션
│   │   └── SaveManager.gd     # 저장/로드
│   ├── Boot.gd                # 시작 화면
│   └── MainGame.gd            # 메인 게임 (종목리스트+매매)
├── scenes/
│   ├── boot.tscn              # 타이틀
│   └── main.tscn              # 메인
└── icon.svg                   # 프로젝트 아이콘
```

### 싱글톤 설계
| 싱글톤 | 역할 | 상태 |
|--------|------|------|
| GameManager | 플레이어 현금/보유/스탯/직급 | ✅ 구현 |
| MarketSim | 주가 시뮬레이션 엔진 | ✅ 구현 |
| SaveManager | JSON 저장/로드 + 오프라인 보상 | ✅ 구현 |
| EventManager | 뉴스/리스크/라이프 이벤트 | 🔲 TODO |
| NPCManager | 호감도/결혼버프/정보거래 | 🔲 TODO |
